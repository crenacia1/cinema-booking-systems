# 🎬 Cinema Seat Booking System

## 📌 Description
A simple web-based cinema booking system that allows users to select movies, view seat availability, and book seats. Each movie has different pricing and the system calculates total cost automatically.

---

## 🎥 Features
- Movie selection with pricing
- Interactive seat booking
- Cinema-style seat layout
- Total price calculation
- Reset seats option
- Works on Netlify (no backend)
